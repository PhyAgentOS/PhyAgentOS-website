仿真环境运行指南

本节聚焦如何在 Isaac Sim 仿真环境中跑通完整的 demo 流程，适用于没有真机或想先在仿真里验证的场景。

① 安装 Isaac Sim 5.1

前往 NVIDIA 官方页面下载并安装 Isaac Sim 5.1.0：

  • 下载页面 (5.1.0)
  • 快速安装指南 (5.1.0)

② 准备 Python 环境

conda activate paos

③ 安装依赖

将 PhyAgentOS 以可编辑模式安装到 paos 环境中：

cd /home/zyserver/work/PhyAgentOS
pip install -e .

④ 启动 HAL Watchdog（GUI 模式）

cd /home/zyserver/work/PhyAgentOS
conda activate paos
python hal/hal_watchdog.py --gui --interval 0.05 \
  --driver pipergo2_manipulation \
  --driver-config examples/pipergo2_manipulation_driver.json
--gui 会打开仿真可视化窗口；--interval 0.05 设置控制循环周期为 50ms，能有效减少仿真卡顿。

⑤ 发送 Agent 指令

打开另一个终端，执行以下命令：

cd /home/zyserver/work/PhyAgentOS
conda activate paos

然后按顺序运行：

# 启动仿真场景
paos agent -m "open simulation"

# 移动到目标位置
paos agent -m "go to desk"

# 执行抓取任务
paos agent -m "pick up the red cube and move next to the rear pedestal"

⑥ 注意事项

单进程

同一时间只保留一个 Watchdog 进程运行，避免端口冲突。

改完重启

修改了 driver 或 skill 文件后，需要重启 Watchdog 才能生效。

卡顿排查

如果仿真画面卡顿，确认启动时使用了 --interval 0.05 参数。
