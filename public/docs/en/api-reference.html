paos agent

Simulation Runtime Guide

This section focuses on how to run the complete demo pipeline in the Isaac Sim simulation environment — ideal if you don't have real hardware or want to verify in simulation first.

① Install Isaac Sim 5.1

Download and install Isaac Sim 5.1.0 from the NVIDIA official pages:

  • Download Page (5.1.0)
  • Quick Install Guide (5.1.0)

② Prepare Python Environment

conda activate paos

③ Install Dependencies

Install PhyAgentOS in editable mode into the paos environment:

cd /home/zyserver/work/PhyAgentOS
pip install -e .

④ Start HAL Watchdog (GUI Mode)

cd /home/zyserver/work/PhyAgentOS
conda activate paos
python hal/hal_watchdog.py --gui --interval 0.05 \
  --driver pipergo2_manipulation \
  --driver-config examples/pipergo2_manipulation_driver.json
--gui opens the simulation visualization window; --interval 0.05 sets the control loop period to 50ms, which effectively reduces simulation lag.

⑤ Send Agent Commands

Open another terminal and run:

cd /home/zyserver/work/PhyAgentOS
conda activate paos

Then execute in order:

# Launch the simulation scene
paos agent -m "open simulation"

# Move to target location
paos agent -m "go to desk"

# Execute grasping task
paos agent -m "pick up the red cube and move next to the rear pedestal"

⑥ Notes

Single Process

Keep only one Watchdog process running at a time to avoid port conflicts.

Restart After Changes

If you modify driver or skill files, restart the Watchdog for changes to take effect.

Lag Troubleshooting

If the simulation is laggy, make sure you launched with --interval 0.05.

Documentation is continuously improving. If you find commands that don't match actual behavior, please open an Issue.
