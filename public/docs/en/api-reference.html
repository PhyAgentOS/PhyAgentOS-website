Three Commands to Get Started

# 1. Initialize workspace
paos onboard

# 2. Start simulator or real robot driver
python hal/hal_watchdog.py --driver simulation

# 3. Enter interactive Agent
paos agent

For more details and troubleshooting, see the Plugin Usage Guide.
